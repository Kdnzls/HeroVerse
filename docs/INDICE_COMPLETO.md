## 💡 Dicas Úteis

### **Enquanto usa a aplicação:**
- Busque "Batman" para teste rápido
- Tente "InvalidHero" para ver tratamento de erro
- Use DevTools (F12) → Network para ver requisições
- Verifique LocalStorage em F12 → Application

### **Enquanto estuda o código:**
- Comece pelo HTML para entender a estrutura
- Depois veja o CSS para estilo
- Por último, estude o JavaScript para lógica
- Use comentários do código como guia

### **Enquanto faz SCRUM:**
- Use README.md como documento de project
- Reference TESTES_E_VALIDACAO.md antes de PR
- Mantenha RESUMO_EXECUTIVO.md atualizado
- Documente mudanças em DOCUMENTACAO_TECNICA.md
